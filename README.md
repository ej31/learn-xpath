# 🕵️ XPath 학습 (Learn XPath)
웹 크롤링(Web Crawling)과 테스트 자동화(Selenium, Playwright 등)의 핵심 기술인 **XPath**를 학습하고 실습할 수 있는 프로젝트입니다.

이 저장소는 이론 학습부터 실전 예제 풀이까지 한 번에 끝낼 수 있도록 구성되어 있습니다.

## 🚀 라이브 데모 (Live Demo)

별도의 설치 없이 웹에서 바로 실습해 보세요!
👉 [**학습 센터 바로가기**](https://ej31.github.io/learn-xpath/)

## 📂 저장소 구성

이 프로젝트는 다음과 같은 학습 자료들을 포함하고 있습니다.

### 1. 📘 이론 가이드 (`xpath_guide.md`)

- XPath의 기초 문법부터 고급 기법까지 정리된 마크다운 문서입니다.
- **포함 내용:** 절대/상대 경로, 속성 선택자, 텍스트 매칭, 축(Axes) 이동, 그리고 실무에서 자주 겪는 함정들.

### 2. 🎮 자유 연습장 (`xpath_practice.html`)

- 배운 문법을 즉시 테스트해 볼 수 있는 인터랙티브 페이지입니다.
- 입력한 XPath에 해당하는 요소가 화면에서 실시간으로 **하이라이트** 됩니다.
- 다양한 HTML 구조(폼, 리스트, 테이블 등)가 준비되어 있습니다.

### 3. 📝 실전 문제집 (`xpath_workbook.html`)

- 실제 쇼핑몰이나 관리자 페이지와 유사한 환경에서 주어진 미션을 수행합니다.
- **Level 1~3**까지 난이도별 문제가 준비되어 있습니다.
- 힌트 기능을 제공하여 스스로 답을 찾도록 유도합니다.

### 4. 🗝️ 정답 및 해설 (`xpath_solutions.md`)

- 실전 문제집의 모범 답안과 상세한 해설이 담겨 있습니다.
- 단순한 정답뿐만 아니라, "**왜 이 XPath가 더 좋은지**"에 대한 팁을 제공합니다.

## 💻 로컬에서 실행하기

이 프로젝트는 순수 HTML/JS로 작성되어 있어 별도의 서버나 설치 과정이 필요 없습니다.

1. 저장소를 클론(Clone)하거나 ZIP 파일로 다운로드합니다.
    
    ```
    git clone https://github.com/ej31/learn-xpath.git
    ```
    
2. 다운로드한 폴더 내의 `index.html` 파일을 더블 클릭하여 브라우저에서 엽니다.
3. 실습을 시작합니다!
