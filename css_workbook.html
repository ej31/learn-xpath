LEVEL 1. 메뉴 네비게이션

난이도: 하

Mission 1-1

📚 학습 키워드: 클래스 선택자(Class Selector), 붙여쓰기(Chained Classes)

'Active' 상태인 메뉴 아이템을 선택하세요. (홈 메뉴)

클래스 선택자(.)를 사용하세요. 공백 없이 연결하면 AND 조건입니다.

Mission 1-2

📚 학습 키워드: 자손 결합자(Descendant Combinator), 공백(Space)

드롭다운 메뉴 내부의 모든 링크(a 태그)를 선택하세요.

자손 결합자(공백)를 활용하여 상위 ID와 하위 태그를 조합하세요.

  • 홈 (Home)
  • 소개
  • 서비스 (Dropdown)
    웹 개발 앱 디자인

LEVEL 2. 회원가입 폼

난이도: 중

Mission 2-1

📚 학습 키워드: 속성 선택자(Attribute Selector), 대괄호([])

필수 입력(required) 속성이 있는 이메일 입력창을 선택하세요.

속성 선택자 [attribute]를 사용해보세요. type 속성도 함께 체크하면 더 정확합니다.

Mission 2-2

📚 학습 키워드: 인접 형제 결합자(Adjacent Sibling Combinator), 더하기(+) 기호

'비밀번호' 라벨 바로 다음에 오는 입력창을 선택하세요.

인접 형제 결합자(+)를 사용하여 라벨 바로 뒤의 요소를 찾으세요.

LEVEL 3. 상품 목록 관리

난이도: 상

Mission 3-1

📚 학습 키워드: 구조 가상 클래스, :nth-child()

목록의 세 번째 상품(Item 3)만 정확히 선택하세요.

:nth-child(n) 가상 클래스를 사용하세요.

Mission 3-2

📚 학습 키워드: 부정 가상 클래스, :not(), :disabled

'품절' 상태가 아닌(구매 가능한) 모든 버튼을 선택하세요.

:not() 가상 클래스를 사용하여 disabled 속성이 없거나, sold-out 클래스가 없는 요소를 찾으세요.

  • Item 1
  • Item 2
  • Item 3 (Target)
  • Item 4

LEVEL 4. 파일 탐색기

난이도: 최상

Mission 4-1

📚 학습 키워드: 자식 결합자(Child Combinator), 꺾쇠(>) 기호

'My Documents' 폴더 바로 아래에 있는 파일들만 선택하세요. (하위 폴더 속의 파일은 제외)

직계 자식 결합자(>)를 사용하여 깊이를 제한하세요.

Mission 4-2

📚 학습 키워드: 일반 형제 결합자(General Sibling), 물결(~) 기호

'Active' 파일 이후에 나오는 모든 파일을 선택하세요.

일반 형제 결합자(~)를 사용하면 기준점 이후의 형제들을 모두 선택할 수 있습니다.

📁 My Documents
📄 Report.docx (Target)
📄 Budget.xlsx (Active)
📄 Photo.jpg (Target)
📁 Old Work
📄 Backup.zip

LEVEL 5. 고급 설정 패널

난이도: 헬

Mission 5-1

📚 학습 키워드: 체크 상태(:checked), 인접 형제(+)

선택된(Checked) 라디오 버튼의 라벨을 선택하세요.

:checked 가상 클래스와 인접 형제 결합자(+)를 조합해보세요.

Mission 5-2

📚 학습 키워드: 타입 순서 가상 클래스, :nth-of-type()

문서 내의 두 번째 문단(p 태그)을 선택하세요. (중간에 다른 태그가 섞여 있어도 p 태그끼리만 순서를 세야 함)

:nth-child 대신 :nth-of-type을 사용해야 합니다.
